    习中特思想

    “21世纪的马克思主义”

    一些经验

    • 重点和主体 ——> 政府
    • 保障 ——> 制度、领导
    • 基础 ——> 农业（物质）
    • 行动指南 ——> 理论

    五位一体：经济建设，政治建设，文化建设，社会建设，生态文明建设

    四个伟大：伟大梦想，伟大工程，伟大斗争，伟大事业

    十个明确，十四个坚持，六个必须坚持（明确作为引领）

    四个全面：全面建设社会主义现代化国家（引领），全面深化改革，全面依法治国，全面从严治党

    四个自信：制度自信、文化自信、道路自信、理论自信

    五位一体

    经济建设，政治建设，文化建设，社会建设，生态文明建设

    高质量发展，新发展格局，建设社会主义现代化国家

    • 最终目的：人民幸福安康
    • 基点：加速构建新发展格局
    • 必由之路：高水平科技自立自强
    • 根据：农业强国
    • 必然要求：农业现代化

    新安全格局的主阵地：维护重点领域国家安全

    社会主义价值观和社会主义价值体系，社会主义价值观体现：先进性，人民性，真实性

    党的根本政治立场：人民立场

    四个全面

    全面建设社会主义现代化国家，全面深化改革，全面依法治国，全面从严治党

    四个全面：全面建设社会主义现代化国家（引领）；全面深化改革；全面依法治国；全面从严治党

    权利的法律保证：立法、宪法、行政保障

    从严治党，反复鲜明特征：标本兼治

    四个自信

    制度自信、文化自信、道路自信、理论自信

    制度自信：制度自信首先是政治制度自信

    文化自信

    • 推动文化繁荣的关键
    • 首要任务：立足中华民族伟大历史实践和当代实践，用中国道理总结好中国经验，把中国经验提升为中国理论，既不盲从各种教条，也不照搬外国理论，实现精神上的独立自主

    文化的：连续性、包容性、统一性

    • 连续性决定了：走自己的路
    • 包容性决定了：中华民族交往交流交融的历史取向，决定了中国各宗教信仰多元并存的和谐格局，决定了中华文化对世界文明兼收并蓄的开放胸怀
    • 统一性决定了：中华民族各民族文化融为一体

    马克思主义是党

    • 把握历史的根本所在
    • 鲜明的政治品格和政治优势

    党全部理论和实践的立足点：走自己的路（文化的连续性决定我们必须走自己的路）

    坚持自信自立，突出强调和体现的是：将尊重客观规律和激发创新活力贯通起来（守正创新）

    法治建设

    建设社会主义法治国家核心要义

    • 坚持党的领导
    • 坚持中国特色社会主义制度
    • 贯彻中国特色社会主义法治理论

    科学立法、严格执法、公正司法、全民守法是新时代我国法治建设的指导方针，并不是核心要义

    全面依法治国的

    • 前提：科学立法
    • 基础：全民守法
    • 关键：严格执法
    • 重点：司法
    • 指南、根本遵循：习近平新时代法治思想
    • 总抓手：建设法制社会体系

    建设社会主义法治国家，首要任务是：全面贯彻实施宪法

    思想道德修养

    爱国主义教育主题：全面建成社会主义现代化国家，实现中华民族伟大复兴

    发展教育的

    • 根本尺度：人民满意
    • 根本目标：为党育人，为国育才

    交往对社会生活的影响

    • 文化发展传播
    • 社会关系进步
    • 生产力发展
    • 人全面发展

    意识形态

    • 关乎旗帜、道路
    • 关乎国家政治安全
    • 决定文化前进道路和发展方向

    个人理想和社会理想
