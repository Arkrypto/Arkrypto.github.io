    一阶微分方程

    什么是微分方程

    微分方程：顾名思义，就是含有微分的方程，其自变量为x，当然，高阶微分方程的自变量为矩阵，要解的结果为微分的原函数，即求出能令其微分满足方程等式的函数

    这里不可避免得要涉及到积分（因为要求原函数），在无条件时求出来的原函数，叫做微分方程的通解（可以理解为不定积分结果，存在任意常数 C）；在给定的初值条件下，可以确定任意常数 C，从而得到一个确定的原函数（定积分），叫做微分方程在初值条件下的特解

    变量可分离的一阶方程

    形如

    y′=f(x)g(y) y' = f(x)g(y) y′=f(x)g(y)
    的微分方程（这里 y 是一个关于 x 的一元函数），可以拆解为两个积分的等式
    dyg(y)=f(x)dx \frac{dy}{g(y)} = f(x)dx g(y)dy​=f(x)dx
    于是我们可以对方程两侧同时积分，从而解出原函数y(x)

    齐次微分方程

    齐次指因变量y和自变量x的指数齐次，如y/x, y^2/x^2就是齐次的

    令y = ux，让原微分方程的变量由x,y变为x,u，从而让该齐次微分方程转变为一个可分离变量的微分方程，然后积分求解

    其中，u也是一个关于x的函数，有

    y′=(ux)′=u′x+u y' = (ux)'= u'x + u y′=(ux)′=u′x+u
    如对于齐次方程
    y′=yx+tanyx y' = \frac{y}{x} + tan\frac{y}{x} y′=xy​+tanxy​
    显然，y 和 x 耦合在一起，不可分离，于是我们令y = ux，则原式为
    u′x+u=u+tanu⇒u′x=tanu u'x + u = u + tanu \Rightarrow u'x = tanu u′x+u=u+tanu⇒u′x=tanu
    此时，神奇的事情发生了，y 和 x 成功解耦，分离得
    dutanu=dxx \frac{du}{tanu} = \frac{dx}{x} tanudu​=xdx​
    积分求解得（tanu分之一是cosu/sinu）
    ∫cosusinudu=∫1xdx⇒ln∣sinu∣=ln∣x∣+C1 \int \frac{cosu}{sinu}du = \int \frac{1}{x}dx \\ \Rightarrow ln|sinu| = ln|x| + C_1 ∫sinucosu​du=∫x1​dx⇒ln∣sinu∣=ln∣x∣+C1​
    指得
    sinu=eln∣x∣+C1=eln∣x∣eC1=xeC1⇒u=arcsinxeC1 sinu = e^{ln|x|+C_1} = e^{ln|x|} e^{C_1} = xe^{C_1}\\ \Rightarrow u = arcsin\,xe^{C_1} sinu=eln∣x∣+C1​=eln∣x∣eC1​=xeC1​⇒u=arcsinxeC1​
    最后代回y（要求得本来就是函数y），u只是工具人，y = ux
    y=xarcsinxeC y = xarcsin\,xe^{C} y=xarcsinxeC
    得到原齐次方程的通解，若告知初值条件，可解得特解，如
    y(1)=arcsineC=π2⇒C=0 y(1) = arcsin\,e^C = \frac{\pi}{2} \Rightarrow C = 0 y(1)=arcsineC=2π​⇒C=0
    故在初值条件y(1) = π/2下的特解为
    y=xarcsinx y = xarcsinx y=xarcsinx

    一阶线性微分方程

    形如

    y′+p(x)y=q(x) y' + p(x)y = q(x) y′+p(x)y=q(x)
    的微分方程，称为一阶线性微分方程，有公式
    y=e−∫p(x)dx[∫q(x)e∫p(x)dxdx+C] y = e^{-\int p(x)dx}[\int q(x)e^{\int p(x)dx}dx + C] y=e−∫p(x)dx[∫q(x)e∫p(x)dxdx+C]
    其中 C 为任意常数，以此解出原微分方程的通解，再通过初值条件可解出特解，这个吊公式怎么推的不知道，有时间再看看

    伯努利方程

    形如

    y′+p(x)y=q(x)yn y' + p(x)y = q(x)y^n y′+p(x)y=q(x)yn
    可以化为
    y−ndydx+p(x)y1−n=q(x) y^{-n}\frac{dy}{dx} + p(x)y^{1-n} = q(x) y−ndxdy​+p(x)y1−n=q(x)
    就是一阶线性微分方程的特殊情况，在微分方程中，存在y的两个幂，且其指数只相差1，此时可以用高阶的y的微分代替低阶的y
    dyn=nyn−1dydxdx=nyn−1dy⇒yn−1=dynndy d{y^{n}} = ny^{n-1}\frac{dy}{dx}dx = ny^{n-1}dy\\ \Rightarrow y^{n-1} = \frac{dy^n}{ndy} dyn=nyn−1dxdy​dx=nyn−1dy⇒yn−1=ndydyn​
    带入伯努利方程
    dy1−n(1−n)dydydx+p(x)y1−n=q(x) \frac{dy^{1-n}}{(1-n)dy}\frac{dy}{dx} + p(x)y^{1-n} = q(x) (1−n)dydy1−n​dxdy​+p(x)y1−n=q(x)
    化简为
    dy1−n(1−n)dx+p(x)y1−n=q(x) \frac{dy^{1-n}}{(1-n)dx} + p(x)y^{1-n} = q(x) (1−n)dxdy1−n​+p(x)y1−n=q(x)
    令y^{1-n} = z，则dz/dx = z'，原式为
    z′(1−n)+p(x)z=q(x) \frac{z'}{(1-n)} + p(x)z = q(x) (1−n)z′​+p(x)z=q(x)
    是不是很眼熟，就是一个一阶线性微分方程，然后就可以代公式
    y=e−∫p(x)dx[∫q(x)e∫p(x)dxdx+C] y = e^{-\int p(x)dx}[\int q(x)e^{\int p(x)dx}dx + C] y=e−∫p(x)dx[∫q(x)e∫p(x)dxdx+C]
    注意原公式的y'前面是没有多余系数的，就是要把那个1-n乘上去系数化为1，然后才能无脑代公式

    全微分方程

    形如

    du(x,y)=P(x,y)dx+Q(x,y)dy=0 du(x,y) = P(x,y)dx + Q(x,y)dy = 0 du(x,y)=P(x,y)dx+Q(x,y)dy=0
    的二元微分方程，称作全微分方程，要求的解为原函数u(x,y)，其中
    P(x,y)=∂u∂xQ(x,y)=∂u∂y P(x,y) = \frac{∂u}{∂x}\quad Q(x,y) = \frac{∂u}{∂y} P(x,y)=∂x∂u​Q(x,y)=∂y∂u​
    这个方程成立的充要条件是
    ∂P∂y=∂Q∂x \frac{∂P}{∂y} = \frac{∂Q}{∂x} ∂y∂P​=∂x∂Q​
    即二阶偏导相同
    ∂2u∂x∂y=∂2u∂y∂x \frac{∂^2u}{∂x∂y} = \frac{∂^2u}{∂y∂x} ∂x∂y∂2u​=∂y∂x∂2u​
    以此可以消除某些变量，再通过
    ∂u∂x=P(x,y) \frac{∂u}{∂x} = P(x,y) ∂x∂u​=P(x,y)
    对方程两边x不定积分得到一个u的变式，记作u0，再对这个u0
    ∂u0∂y=Q(x,y) \frac{∂u_0}{∂y} = Q(x,y) ∂y∂u0​​=Q(x,y)
    消除所有变量，得到最终的通解u(x,y)

    二阶及高阶线性微分方程

    自此发现，不定积分其实是微分方程的一个子集，实际上就是缺省中间阶导数的微分方程，而定积分就是在不定积分的基础上应用牛顿莱布尼兹公式所得

    题目的侧重不同，一个重积分过程，一个求解

    线性微分方程及线性无关

    n 阶线性微分方程

    y(n)+a1y(n−1)+...+an−1y′+any=f(x) y^{(n)}+a_1y^{(n-1)}+...+a_{n-1}y'+a_ny = f(x) y(n)+a1​y(n−1)+...+an−1​y′+an​y=f(x)
    其中f(x)为自由项

    • 自由项为 0，该线性微分方程为齐次
    • 否则，该线性微分方程非齐次

    线性相关和无关

    k1y1(x)+k2y2(x)+...+knyn(x)=0 k_1y_1(x)+k_2y_2(x)+...+k_ny_n(x)=0 k1​y1​(x)+k2​y2​(x)+...+kn​yn​(x)=0
    若存在不全为 0 的系数组合ki使得等式成立，则说明yi(x)是线性无关的，否则我们说yi(x)线性相关

    线性无关也意味着任意两个项的商不为常数，即

    kiyi(x)kjyj(x)≠C∈R,i≠j \frac{k_iy_i(x)}{k_jy_j(x)} \neq C\in R\,\,\,,i\neq j kj​yj​(x)ki​yi​(x)​=C∈R,i=j

    齐次线性微分方程

    齐次线性微分方程解的性质：解和解的线性相关项仍是解，譬如y1,y2是齐次线性方程的解，则

    y3=y1+y2y4=y3−4y1+y2 y_3 = y_1+y_2\quad y_4 = y_3-4y_1+y_2 y3​=y1​+y2​y4​=y3​−4y1​+y2​
    y3,y4仍是原方程的解，当然y3,y4是线性相关的

    齐次线性微分方程的通解：其中yi均线性无关

    Y(x)=∑i=1mCiyi Y(x) = \sum_{i=1}^{m}C_iy_i Y(x)=i=1∑m​Ci​yi​
    求解二阶齐次线性微分方程的通解，对于方程
    y′′+py′+q=0 y'' + py' + q = 0 y′′+py′+q=0
    有其特征方程
    r2+pr+q=0 r^2 + pr + q = 0 r2+pr+q=0
    这是一个一元二次方程，通过公式求解可得解r，根据r的不同，原方程的通解可分为
    y={C1er1x+C2er2xr=r1/r2(双根)(C1+C2x)er0xr=r0(重根)eax(C1cosbx+C2sinbx)r=a±bi(复数根) y = \begin{cases} C_1e^{r_1x}+C_2e^{r_2x}\quad r = r_1/r_2(双根)\\ (C1+C2x)e^{r_0x}\quad r = r_0(重根)\\ e^{ax}(C_1cosbx + C_2sinbx)\quad r = a\pm bi(复数根) \end{cases} y=⎩⎨⎧​C1​er1​x+C2​er2​xr=r1​/r2​(双根)(C1+C2x)er0​xr=r0​(重根)eax(C1​cosbx+C2​sinbx)r=a±bi(复数根)​
    所谓非齐次微分线性方程对应的齐次微分线性方程就是把非齐次方程的自由项手动化为 0 的齐次方程

    非齐次线性微分方程

    非齐次线性微分方程的解 = 对应齐次方程的通解 + 非齐次方程的一个特解

    y=Y(x)+y∗(x) y = Y(x) + y^*(x) y=Y(x)+y∗(x)
    自然，也可以视作 n 个线性无关的齐次方程解加上一个非齐次方程特解
    y=∑i=1mCiyi+y∗(x) y = \sum_{i=1}^{m}C_iy_i + y^*(x) y=i=1∑m​Ci​yi​+y∗(x)
    特解的求法使用猜测法，即猜测特解的形式设常系数，再带入原方程求解常系数最终得到特解，如对于非齐次微分方程
    y′′+y′+4y=3x y'' + y' +4y = 3x y′′+y′+4y=3x
    则设其特解形式为
    y∗=ax+b y^* =ax + b y∗=ax+b
    解叠加原理：对于非齐次线性方程
    y(n)+a1y(n−1)+...+an−1y′+any=f(x)y(n)+a1y(n−1)+...+an−1y′+any=g(x) y^{(n)}+a_1y^{(n-1)}+...+a_{n-1}y'+a_ny = f(x) \\ y^{(n)}+a_1y^{(n-1)}+...+a_{n-1}y'+a_ny = g(x) y(n)+a1​y(n−1)+...+an−1​y′+an​y=f(x)y(n)+a1​y(n−1)+...+an−1​y′+an​y=g(x)
    若已知其特解分别为y1*和y2*

    则对于自由项为两个方程自由项之和的非齐次线性微分方程

    y(n)+a1y(n−1)+...+an−1y′+any=f(x)+g(x) y^{(n)}+a_1y^{(n-1)}+...+a_{n-1}y'+a_ny = f(x) + g(x) y(n)+a1​y(n−1)+...+an−1​y′+an​y=f(x)+g(x)
    有特解为
    y∗=y1∗+y2∗ y^* = y_1^* + y_2^* y∗=y1∗​+y2∗​
    下面开始抽象起来了

    差分方程

    差分方程概念

    什么是差分，对于数列an

    an+1−anan−1−an−3 a_{n+1}-a_n\quad a_{n-1}-a_{n-3} an+1​−an​an−1​−an−3​
    就是关于an的两个差分

    对于函数而言同理，如函数f(x)

    f(x+1)−f(x) f(x+1)-f(x) f(x+1)−f(x)
    便是他的一个差分

    而差分方程，就是有以函数差分为项的方程，如

    f(x+1)+af(x)=g(x) f(x+1) + af(x) = g(x) f(x+1)+af(x)=g(x)
    其中g(x)为一个已知函数，差分方程的解即为使得方程成立的函数f(x)

    和微分方程类似

    • 当g(x) = 0时，差分方程为齐次
    • 否则为非齐次

    求解差分方程

    对于二阶非齐次差分方程

    f(x+1)+af(x)=g(x) f(x+1) + af(x) = g(x) f(x+1)+af(x)=g(x)
    其通解结构为
    y=y∗(x)+Y(x) y = y^*(x) + Y(x) y=y∗(x)+Y(x)
    其中y*为非齐次方程的一个特解，Y为对应非齐次方程的通解

    1、首先求对应齐次方程的通解

    令自由项为 0 得相应齐次差分方程，有特征方程

    r2+ar=0 r^2 + ar = 0 r2+ar=0
    易解得r = -a，所以齐次差分方程通解为
    Y(x)=C(r)x=C(−a)r Y(x) = C(r)^x = C(-a)^r Y(x)=C(r)x=C(−a)r
    2、再求非齐次方程的一个特解

    观察g(x)的形状，以这种视角看待

    g(x)=uxPm(x) g(x) = u^xP_m(x) g(x)=uxPm​(x)
    其中u为常数，Pm(x)为一个关于 x 的最高 m 次方的多项式，和微分方程一样，这里的 m 次和差分方程的阶数保持一致，这里是二阶，m 即为 2，最高次为x^2

    根据比较u和特征解r，设特解形式

    y∗={uxQm(x)u≠rxuxQm(x)u=r y^*= \begin{cases} u^xQ_m(x)\quad u\neq r\\ xu^xQ_m(x)\quad u = r \end{cases} y∗={uxQm​(x)u=rxuxQm​(x)u=r​
    再代入原先的差分方程，根据比较x前系数确定多项式Qm(x)的未知系数，得到完整的没有未知数的特解y*
    y∗(x+1)+ay∗(x)=g(x) y^*(x+1) + ay^*(x) = g(x) y∗(x+1)+ay∗(x)=g(x)
    3、最后齐次通解加非齐次特解得到非齐次通解
    y=y∗+Y y = y^* + Y y=y∗+Y
    解毕
