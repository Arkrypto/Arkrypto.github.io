    试试你东b的离散书

    集合是数据，关系是逻辑，函数是不允许一对多的关系（一种较为特殊的逻辑），数理逻辑是基础

    数理逻辑

    命题、连结词、命题公式

    命题

    连结词：非、析取、合取、蕴含、等价

    命题公式：永真式、矛盾式、可满足式

    命题公式的转换

    等值公式：德摩根式（析取和合取的转化）

    化范式的常见步骤

    • 通过德摩根律将所有连结词化为：析取、合取、非
    • 通过分配律和结合律，将析取/合取小项结合

    范式：合取范式、析取范式

    推理规则

    真值表法

    直接、间接证明法：推理的前提是条件永真

    • P 为条件
    • T(1)(2) 表示当前命题公式（永真）由条件 1、2 推理得到

    附加前提证明法（CP 规则）

    A∧B⇒C→D A\wedge B\Rightarrow C\rightarrow D A∧B⇒C→D
    等价于推理
    A∧B∧C⇒D A\wedge B\wedge C\Rightarrow D A∧B∧C⇒D
    此时有附加条件 C 为永真，只用推理出 D 永真，即可证明C->D永真

    谓词和量词

    量词：全称量词（所有），存在量词（存在）

    ¬∀xF(x)⟺∃x¬F(x) \neg \forall xF(x) \iff \exists x\neg F(x) ¬∀xF(x)⟺∃x¬F(x)

    集合

    集合、子集、幂集

    基本概念

    • 集合
    • 元素
    • 有限集
    • 无限集

    集合的表示法

    • 穷举法（列举法）
    • 描述法

    空集：自身不包含任何元素的集合，空集 ∅ 是唯一的，为任何集合的子集

    ∅={} \empty = \{\} ∅={}
    元素、集合的关系（包含的关系，箭头指向谁，谁被包含）

    • 属于：记作 a ∈ A，即元素 a 属于集合 A（集合 A 中包含元素 a）
    • 相等：两个集合相等当且仅当他们有相同的元素
    • 子集：记为 ⊃，A ⊃ B 表示 A 包含 B（B 是 A 的子集），具有自反性、传递性

    当两个集合互为对方的子集时，两集合相等，空集为任何集合的子集

    注意属于和包含的符号，箭头朝哪边，哪个集合作宾语，即被包含的一方

    子集与幂集

    • 幂集：A 的幂集记为 P(A)，P(A) 为“子集的集合”，其元素为集合 A 的所有子集，换句话说，P(A) 是 A 的所有子集组成的集合

    一个集合 A，其包含 k 个元素的子集的个数有

    Cnk C_n^k Cnk​
    故 A 的幂集 P(A) 元素个数为
    1+∑k=1nCnk=1+Cn1+...+Cnn=2n 1+\sum_{k=1}^nC_n^k = 1+C_n^1+...+C_n^n=2^n 1+k=1∑n​Cnk​=1+Cn1​+...+Cnn​=2n
    即为 A 的子集的个数，其中 1 为空集

    集合运算

    集合的基本运算及性质：交与并，差与补

    差的常用定理（结合概率论理解）

    A−B=A∩B‾=A−(A∩B) A-B=A∩\overline{B}=A-(A∩B) A−B=A∩B=A−(A∩B)
    注意补是参照于全集的概念，如全集为 E，则 A 的补为
    A‾=E−A \overline{A} = E-A A=E−A

    德摩根律：并的非 = 非的交

    A∪B‾=A‾∩B‾ \overline{A∪B}=\overline{A}∩\overline{B} A∪B=A∩B
    交的非 = 非的并
    A∩B‾=A‾∪B‾ \overline{A∩B}=\overline{A}∪\overline{B} A∩B=A∪B

    这和数理逻辑中合取析取的德摩根律是一样的

    ¬(A∧B)=¬A∨¬B \lnot(A\wedge B)=\lnot A\vee \lnot B ¬(A∧B)=¬A∨¬B
    对称差，用异或符号表示
    A⊕B=(A−B)∪(B−A) A\oplus B=(A-B)∪(B-A) A⊕B=(A−B)∪(B−A)
    其中差运算可替换为
    A−B=A∩B‾B−A=B∩A‾ A-B = A∩\overline{B}\quad B-A=B∩\overline{A} A−B=A∩BB−A=B∩A
    显然
    A⊕A=(A−A)∪(A−A)=∅A⊕∅=(A−∅)∪(∅−A)=A A\oplus A = (A-A)∪(A-A)=\empty\\ A\oplus\empty=(A-\empty)∪(\empty-A)=A A⊕A=(A−A)∪(A−A)=∅A⊕∅=(A−∅)∪(∅−A)=A
    且满足结合律
    A⊕B⊕C=A⊕(B⊕C) A\oplus B\oplus C = A\oplus (B\oplus C) A⊕B⊕C=A⊕(B⊕C)
    有点像概率里的加法（只是形式上，意义好像根本不搭嘎）
    A⊕B⊕C=A+B+C−AB−AC−BC+ABC A⊕B⊕C = A+B+C-AB-AC-BC+ABC A⊕B⊕C=A+B+C−AB−AC−BC+ABC

    包含排斥原理，讨论有限集中元素的计数问题

    比如

    A∩B=∅⇒∣A∣+∣B∣=∣A∪B∣ A∩B=\empty\Rightarrow |A|+|B|=|A∪B| A∩B=∅⇒∣A∣+∣B∣=∣A∪B∣
    其中 |A| 表示集合 A 中元素的数量

    参考概率论的包含排斥关系，包含排斥原理为

    ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣ |A∪B| = |A|+|B|-|A∩B| ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣

    可以推广到多个集合

    ∣A+B+C∣=∣A∣+∣B∣+∣C∣−∣AB∣−∣AC∣−∣BC∣+∣ABC∣ |A+B+C|=|A|+|B|+|C|-|AB|-|AC|-|BC|+|ABC| ∣A+B+C∣=∣A∣+∣B∣+∣C∣−∣AB∣−∣AC∣−∣BC∣+∣ABC∣

    二元关系

    关系的性质

    序偶和笛卡尔乘积

    • 序偶即有序的两个元素对
    • 笛卡尔乘积

    关系及其表示：关系是逻辑，作用于集合之上才有意义

    关系的性质：自反、传递、对称、反对称

    • 均可以用关系矩阵来判断其性质

    复合关系和逆关系

    • 复合关系：即合成关系，通过中间命题传递，在关系矩阵中表现为矩阵乘法
    • 逆关系：逻辑不变，序偶顺序改变，在关系矩阵中表现为矩阵转置

    关系的闭包运算：自反闭包（r）、传递闭包（t）、对称闭包（s）

    等价关系和等价类：自反性、传递性、对称性

    偏序集

    具有自反性、传递性、反对称性的二元关系集合（如整数集合上的小于等于关系）

    偏序集中的大小比较

    • 注意能比较的前提是两元素具有偏序关系，所以孤立点不比任何元素大，也不比任何元素小，进而有孤立点的偏序集必定没有最大元和最小元

    函数

    函数的性质

    函数是一种特殊的二元关系，他要求二元关系的前件集合 A 中的每个元素 x 在后件集合 B 中都有唯一一个元素 y 与之构成属于该函数的序偶

    • 就是说，一个 x 只能对应一个 f(x)，不能一对多，如{<1,2>, <1,3>}就是不被允许的

    单射：若 x1 ≠ x2，则 f(x1) ≠ f(x2)，即每个存在的 y 对应唯一的一个 x（单调函数就是典型的单射，但注意单射函数不一定单调），当然在后件集合 B 中允许有不被 x 映射的 y

    满射：函数的值域为后件集合，即后件 B 被映射满了

    双射：同时满足单射和漫射，此时每一个 x 和 y 一一对应（具有反函数）

    函数运算

    复合运算：和关系的复合运算一样，就是复合函数

    f(x)og(x)=g(f(x)) f(x)\,o\,g(x) = g(f(x)) f(x)og(x)=g(f(x))
    逆运算（反函数）：只有双射的函数在经过逆运算之后才有反函数，若非双射，逆运算后会是一个普通的二元关系（而非函数）

    群

    群的不同阶段 性质
    广群 封闭性
    半群 封闭性，结合律
    含幺半群（独异点） 封闭性，结合律，有幺元
    群 封闭性，结合律，有幺元，有逆元
    阿贝尔群 封闭性，结合律，有幺元，有逆元，交换律

    注意，群一定没有零元：若含幺半群有零元，则零元没有逆元，则独异点构不成群

    如在整数集上的乘法，显然 0 是零元，1 是幺元，而 0 * x 恒等于 0 无法得到幺元，即：零元不可能有逆元，故无法构成群

    图论

    图的表示

    邻接矩阵（和 DS 一样）

    • 若上半部分均为 1，则单向连通
    图的分类
    平凡图 由一个孤立点构成的图
    简单图 无平行边，无环的图
    完全图 任意两节点都有边相连的简单图
    连通图 对于有向图，分为单向连通（图中每两个结点可以通过一个方向可达）和强连通
    欧拉图 有欧拉回路的连通无向图
    平面图 在一个平面上，无向图 G 的图解中若没有任何边的交叉，则称图 G 是个平面图；否则，称 G 是非平面图

    通路与回路

    欧拉回路：图 G 上有一条经过所有顶点、所有边的简单回路

    • 简单回路：边不重复，点可以重复
    • 基本回路：边可以重复，点不重复

    路的长度：经过的边的个数

    若题目只说通路和回路，则认为没有不可重复边和结点的限制，只关注起点和终点

    图的性质

    握手定理：任意图的总度数是边数的两倍

    欧拉定理：对于联通平面图，其结点数 v，边数 e，面数 r 满足 v-e+r = 2
