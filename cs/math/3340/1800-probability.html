随机事件和概率

不相容和独立的区别，不相容指二者不能同时发送

P(AB)=0 P(AB) = 0 P(AB)=0
独立指二者是否发生不对另一个事件产生任何影响
P(A∣B)=P(A)P(AB)=P(A)P(B) P(A|B) = P(A) \quad P(AB) = P(A)P(B) P(A∣B)=P(A)P(AB)=P(A)P(B)

对于 17 题，显而易见，条件概率满足

P(A‾∣B‾)=P(A‾∣B) P(\overline{A}|\overline{B}) = P(\overline{A}|B) P(A∣B)=P(A∣B)
根据题意有
P(A‾∣B‾)=1−P(A∣B)=P(A‾∣B) P(\overline{A}|\overline{B}) = 1-P(A|B) = P(\overline{A}|B) P(A∣B)=1−P(A∣B)=P(A∣B)
所以有
1=P(A∣B)+P(A‾∣B) 1 = P(A|B) + P(\overline{A}|B) 1=P(A∣B)+P(A∣B)
故 B 是否发生对 A 根本不产生任何影响，故二者独立

独立性的判断，当二维随机变量边际密度函数满足

pX(x)pY(y)=p(x,y) p_X(x)p_Y(y)=p(x,y) pX​(x)pY​(y)=p(x,y)
则 X 和 Y 互相独立

试算法判断独立性，如对于均匀分布 X~U(0,2)，随机变量 Y = X^2，则

P(X<1,Y<1)=P(X<1)=12 P(X<1,Y<1) = P(X<1) = \frac{1}{2} P(X<1,Y<1)=P(X<1)=21​
但有
P(X<1)P(Y<1)=12×12=14 P(X<1)P(Y<1) = \frac{1}{2}\times\frac{1}{2}= \frac{1}{4} P(X<1)P(Y<1)=21​×21​=41​
故二者不独立（只有联合概率等于编辑概率相乘，两随机变量才互相独立）

方差、协方差和相关系数

方差的运算

D(X+Y)=D(X)+D(Y)+2ρD(X)D(Y) D(X+Y) = D(X)+D(Y)+2\rho\sqrt{D(X)D(Y)} D(X+Y)=D(X)+D(Y)+2ρD(X)D(Y)​
其中
2ρD(X)D(Y)等价于2Cov(X,Y) 2\rho\sqrt{D(X)D(Y)}\quad 等价于\quad 2Cov(X,Y) 2ρD(X)D(Y)​等价于2Cov(X,Y)
协方差
Cov(U,V)=E(UV)−E(U)E(V) Cov(U,V) = E(UV) - E(U)E(V) Cov(U,V)=E(UV)−E(U)E(V)
相关系数
ρ=Cov(U,V)D(U)D(V) \rho = \frac{Cov(U,V)}{\sqrt{D(U)D(V)}} ρ=D(U)D(V)​Cov(U,V)​
随机变量 X 自身对自身的相关系数为 1，协方差为 D(X)
Cov(X,X)=E(X2)−E2(X)=D(X)ρ=1 Cov(X,X) = E(X^2)-E^2(X) = D(X) \quad \rho = 1 Cov(X,X)=E(X2)−E2(X)=D(X)ρ=1
互相独立的变量 X 和 Y 的相关系数为 0
Cov(X,Y)=E(XY)−E(X)E(Y)=0 Cov(X,Y) = E(XY)-E(X)E(Y) = 0 Cov(X,Y)=E(XY)−E(X)E(Y)=0
任意随机变量与常数的相关系数为 0
Cov(X,C)=E(XC)−E(X)E(C)=CE(X)−CE(X)=0 Cov(X, C) = E(XC)-E(X)E(C) = CE(X)-CE(X) = 0 Cov(X,C)=E(XC)−E(X)E(C)=CE(X)−CE(X)=0
线性相关的两个随机变量相关系数一定为 1，设 Y = 2X+1
Cov(X,Y)=E(2X2+X)−E(X)E(2X+1) Cov(X,Y) = E(2X^2+X)-E(X)E(2X+1) Cov(X,Y)=E(2X2+X)−E(X)E(2X+1)
拆开为
2E(X2)+E(X)−E(X)[2E(X)+1]=2E(X2)−2E2(X)=2D(X) 2E(X^2)+E(X)-E(X)[2E(X)+1] = 2E(X^2)-2E^2(X) = 2D(X) 2E(X2)+E(X)−E(X)[2E(X)+1]=2E(X2)−2E2(X)=2D(X)
同时有
D(X)D(Y)=D(X)D(2X+1)=4D2(X)=2D(X) \sqrt{D(X)D(Y)} = \sqrt{D(X)D(2X+1)} = \sqrt{4D^2(X)} = 2D(X) D(X)D(Y)​=D(X)D(2X+1)​=4D2(X)​=2D(X)
故有 ρ = 1（分子分母恒等）

协方差的分配律

Cov(X+2Y,X−2Y) Cov(X+2Y,X-2Y) Cov(X+2Y,X−2Y)
和乘法分配律一模吊样，等于
Cov(X,X)+Cov(X,−2Y)+Cov(2Y,X)+Cov(2Y,−2Y) Cov(X,X)+Cov(X,-2Y)+Cov(2Y,X)+Cov(2Y,-2Y) Cov(X,X)+Cov(X,−2Y)+Cov(2Y,X)+Cov(2Y,−2Y)
这里的系数都可以提出来，直接提，因为协方差的单个变量以期望的形式参与运算（期望的系数提出均为线性，方差为平方），进一步为（中间项约掉）
Cov(X,X)−4Cov(Y,Y) Cov(X,X)-4Cov(Y,Y) Cov(X,X)−4Cov(Y,Y)
故最后化为 D(X)-4D(Y)

不相关和独立

不相关即指，两随机变量相关系数 ρ = 0，满足

Cov(X,Y)=E(XY)−E(X)E(Y)=0 Cov(X,Y) = E(XY)-E(X)E(Y) = 0 Cov(X,Y)=E(XY)−E(X)E(Y)=0
即
E(XY)=E(X)E(Y) E(XY) = E(X)E(Y) E(XY)=E(X)E(Y)
但这并不能说明变量 X 和 Y 是独立的，我们称 X Y 独立，当且仅当
P(XY)=P(X)P(Y) P(XY) = P(X)P(Y) P(XY)=P(X)P(Y)
即二者联合分布等于两随机变量的边际分布直接相乘，参考多为随机变量的独立性（离散随机变量一个道理）
