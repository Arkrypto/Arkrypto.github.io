随机事件和概率

不相容和独立的区别，不相容指二者不能同时发送

P(AB)=0 P(AB) = 0 P(AB)=0
独立指二者是否发生不对另一个事件产生任何影响
P(A∣B)=P(A)P(AB)=P(A)P(B) P(A|B) = P(A) \quad P(AB) = P(A)P(B) P(A∣B)=P(A)P(AB)=P(A)P(B)

对于 17 题，显而易见，条件概率满足

P(A‾∣B‾)=P(A‾∣B) P(\overline{A}|\overline{B}) = P(\overline{A}|B) P(A∣B)=P(A∣B)
根据题意有
P(A‾∣B‾)=1−P(A∣B)=P(A‾∣B) P(\overline{A}|\overline{B}) = 1-P(A|B) = P(\overline{A}|B) P(A∣B)=1−P(A∣B)=P(A∣B)
所以有
1=P(A∣B)+P(A‾∣B) 1 = P(A|B) + P(\overline{A}|B) 1=P(A∣B)+P(A∣B)
故 B 是否发生对 A 根本不产生任何影响，故二者独立

不相关和独立

不相关即指，两随机变量相关系数 ρ = 0，满足

Cov(X,Y)=E(XY)−E(X)E(Y)=0 Cov(X,Y) = E(XY)-E(X)E(Y) = 0 Cov(X,Y)=E(XY)−E(X)E(Y)=0
即
E(XY)=E(X)E(Y) E(XY) = E(X)E(Y) E(XY)=E(X)E(Y)
但这并不能说明变量 X 和 Y 是独立的，我们称 X Y 独立，当且仅当
P(XY)=P(X)P(Y) P(XY) = P(X)P(Y) P(XY)=P(X)P(Y)
